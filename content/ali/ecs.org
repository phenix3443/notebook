* 新建密钥对
* 新建普通用户
  新建用户，并添加 sudo 权限：
  #+BEGIN_SRC sh
add user develop && usermod -aG sudo develop
  #+END_SRC


  为了能使用 develop 远程登录，需要拷贝授权文件
  #+BEGIN_SRC sh
cp authorized_keys /home/develop/.ssh/ && chown develop:develop /home/develop/.ssh/authorized_keys
  #+END_SRC

  然后就可以向上面那样使用 develop 进行登录了。
