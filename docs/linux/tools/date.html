1 概述

用来打印和设置当前系统时间。

2 使用

date [OPTION]... [+FORMAT]

常用的 OPTION 参数：

  • -d, --date=STRING 显示日期字符串描述时间。
  • -s, --set=STRING 设置系统时间为日期字符串描述的时间。

3 日期字符串

日期字符串是人类可读日期字符串，例如 “Sun，2004 年 2 月 29 日 16:21:42 -0800” 或 “2004-02-29 16:21:42” 甚至 “next Thursday”。日期字符串格式比这里容易记录的更复杂，但在 info 文档中有详细描述。

3.1 当前时间

date

2020年 04月 26日 星期日 10:52:13 CST

3.2 当天起始时间

空字符串表示当天的起始时间。 

date -d ""

2020年 04月 26日 星期日 00:00:00 CST

3.3 显示指定日期

日期字符串可以包含日历日期，时间，时区，星期几， 

date -d "2019-01-01 00:00:00"

2019年 01月 01日 星期二 00:00:00 CST

3.4 相对时间

date -d "-1 day"

2020年 04月 25日 星期六 10:52:13 CST


date -d "+2 hour"

2020年 04月 26日 星期日 12:52:13 CST


date -d "next sunday"

2020年 05月 03日 星期日 00:00:00 CST


date -d "last sunday"

2020年 04月 19日 星期日 00:00:00 CST

3.5 时间戳转为字符串

date -d "@1553693532"

2019年 03月 27日 星期三 21:32:12 CST

4 时间格式示例

常用的 FORMAT:

  • %F full date; same as %Y-%m-%d
  • %R 24-hour hour and minute; same as %H:%M
  • %s seconds since 1970-01-01 00:00:00 UTC，用来打印时间戳
  • %x locale's date representation (e.g., 12/31/99)
  • %X locale's time representation (e.g., 23:13:48)

4.1 指定格式

date +"%F %R:%S"

2020-04-26 10:52:13

4.2 显示为时间戳

date +"%s"

1587869533
